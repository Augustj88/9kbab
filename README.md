## 기능
* [포스트(글쓰기)] 자동 태그 추천
* [회원가입] 숫자인식으로 올바른 사용자인지 판단
* [댓글페이지] 댓글 작성 및 리스트 확인
* [메인페이지] 좋아요
* [마이페이지] 스크랩 된 포스트 확인
